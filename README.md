# NOIR CAFÉ — Landing Page Demo

> 바이브코딩 실습 예제 프로젝트입니다.  
> 다크 프리미엄 컨셉의 스페셜티 카페 랜딩페이지를 HTML + CSS + JS 단일 파일로 구현합니다.

---

## 📌 프로젝트 개요

| 항목 | 내용 |
|---|---|
| 목적 | 카페 브랜드 소개 및 방문 유도 홍보 사이트 |
| 타겟 | 20~40대 잠재 고객 |
| 구성 | HTML + CSS + Vanilla JS 단일 파일 (`index.html`) |
| 반응형 | 모바일 768px 이하 대응 |

---

## 🗂 섹션 구성

1. **Hero** — 브랜드 타이틀 + 슬로건 + CTA 버튼
2. **Our Story** — 브랜드 소개 + 스크롤 카운팅 숫자
3. **Signature Menu** — 대표 메뉴 3가지 (호버 인터랙션)
4. **Our Space** — 공간 분위기 이미지 갤러리
5. **Location** — 구글 지도 + 운영시간
6. **CTA** — 최종 방문 유도

## ✨ 주요 기능

- 스크롤 진입 시 페이드인 애니메이션
- Our Story 숫자 카운팅 인터랙션
- 메뉴 카드 호버 시 설명 슬라이드업
- 스크롤 시 네비게이션 글라스모피즘 전환
- 구글 지도 다크 테마 임베드
- 모바일 햄버거 메뉴

## 🚀 실행 방법

별도 설치 없이 `index.html` 파일을 브라우저에서 바로 열면 됩니다.

```bash
open index.html
```
